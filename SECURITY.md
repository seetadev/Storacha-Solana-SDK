# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in toju, please report it responsibly.

**Email:** security@toju.network

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation plan within 7 days.

## Scope

- Solana program (`solana-programs/programs/toju`)
- Backend server (`server/`)
- SDK packages (`packages/sol`, `packages/fil`)
- Web application (`ui/`)

## Out of Scope

- Storacha network infrastructure
- Solana blockchain itself
- Third-party dependencies (report upstream)
